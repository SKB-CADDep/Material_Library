1. НАЧАЛО РАБОТЫ

1.1. Для начала работы выберите меню "Файл" -> "Открыть директорию...".
1.2. Укажите папку, в которой находятся ваши JSON-файлы материалов.
1.3. В папке с приложением должен находиться файл `application_area.txt`. Этот файл содержит список областей применения (каждая с новой строки) и используется для фильтрации и редактирования материалов. Если файл не найден, вы увидите предупреждение.
1.4. Приложение загрузит все найденные материалы и станет готово к работе.

2. ВКЛАДКА "ПОДБОР МАТЕРИАЛА"

Эта вкладка предназначена только для просмотра и анализа существующих материалов.

2.1. Подбор по температуре:
   - Отфильтруйте материалы по "Области применения" (необязательно).
   - Введите расчетную температуру и нажмите "Рассчитать".
   - В таблице отобразятся значения свойств для каждой категории прочности (КП) при заданной температуре.
   - Значения интерполируются, если точной температуры нет. Если значение вне диапазона, будет показан "-".
   - Для сортировки кликните на заголовок столбца. Для копирования значения — правый клик на ячейке.

2.2. Сравнение материалов (свойства):
   - Отфильтруйте материалы по "Области применения" (необязательно).
   - Выберите свойство из выпадающего списка. Список материалов автоматически отфильтруется, показывая только те, у которых есть данные по этому свойству.
   - В списке материалов выберите один или несколько (кликнув на них мышью).
   - Нажмите "Построить график". На графике отобразится зависимость выбранного свойства от температуры.
   - Доступны инструменты работы с графиком (это стандартные элементы встроенной библиотеки и их функции невозможно изменить):
        - Кнопка "Домой" (домик) вернет изображение в исходное состояние,
        - Кнопки "Стрелка влево / вправо" позволяют отменять или повторять произведенные манипуляции с графиком,
        - Кнопка "Перекрестие" позволяет перемещать область графика для визуального контроля областей,
        - Кнопка "Zoom" (лупа) позволяет приближать (с зажатой ЛКМ) и отдалять (с зажатой ПКМ) изображение на графике,
        - Кнопка "Настройки" позволит настроить отображение графика (не рекомендуем использовать!),
        - Кнопка "Сохранить" (дискета) позволит сохранить изображение графика,
        - При наведении курсора на график в правом нижнем углу отображаются координаты курсора, т.е. по X температура, по Y выбранное свойство,
        - Отображение материалов имеет ограничение по цветовой гамме, не более 10 цветов.

2.3. Сравнение материалов (хим. состав):
   - Отфильтруйте материалы по "Области применения" (необязательно).
   - Выберите один или несколько материалов для сравнения.
   - В таблице отобразится их химический состав по разным источникам.
   - Над таблицей появятся поля для фильтрации по элементам. Введите в них желаемое процентное содержание. Таблица автоматически отсортируется и подсветит строки:
     * Зеленым — полное совпадение со всеми заданными фильтрами.
     * Розовым — несовпадение.
   - При наведении курсора на название элемента в заголовке таблицы появится подсказка о его влиянии на свойства стали.

3. ВКЛАДКА "ДОБАВЛЕНИЕ / РЕДАКТИРОВАНИЕ МАТЕРИАЛА"

Здесь можно создавать новые материалы или изменять существующие. Все изменения происходят во временной копии и не затрагивают файлы до сохранения.

3.1. Редактирование материала:
   - Выберите материал из выпадающего списка вверху вкладки (наведите курсор на наименование материала для подгрузки "Области применения").
   - Внесите изменения на любой из подвкладок ("Общие данные", "Физические свойства" и т.д.).
   - В таблицах:
        - для редактирования ячейки используйте двойной клик,
        - "+" добавляет строку внизу списка,
        - "-" удаляет выбранную строку.
   - В таблицах свойств рядом с данными отображается интерактивный график.
   - Чтобы сохранить изменения в файл, выберите меню "Файл" -> "Сохранить".
   - Чтобы отменить все сделанные изменения и вернуть материал к исходному состоянию, выберите "Файл" -> "Отменить изменения".

3.2. Создание нового материала:
   - Нажмите кнопку "Создать новый".
   - Для заполнения мех. свойств и / или хим. состава необходимо добавить "КП" (категорию прочности) и "Источник" соответственно.
   - Заполните данные на всех вкладках.
   - Для сохранения нового материала используйте "Файл" -> "Сохранить как...". Вам будет предложено выбрать имя и местоположение для нового JSON-файла.

3.3. Логирование изменений:
   - При каждом сохранении (через "Сохранить" или "Сохранить как...") все сделанные изменения автоматически записываются в файл `material_changelog.txt`.
   - Этот файл создается в корневой директории программы и содержит информацию о том, кто, когда и какие поля в каком материале изменил.

4. ВКЛАДКА "РАБОТА С ИСТОЧНИКАМИ"

Этот раздел предназначен для глобального управления источниками данных по всем материалам.

4.1. Просмотр и фильтрация:
   - Вкладка автоматически собирает все уникальные источники данных из всех загруженных материалов.
   - Вы можете отфильтровать список по области применения или по части названия материала.

4.2. Массовая замена:
   - Эта функция позволяет найти и заменить название основного источника сразу во всех файлах.
   - Введите текст, который нужно найти, и текст для замены, затем нажмите "Выполнить замену".
   - ВНИМАНИЕ: Это действие необратимо и напрямую изменяет файлы на диске. Используйте с осторожностью.

4.3. Экспорт списка:
   - Кнопка "Сохранить список источников в файл" позволяет экспортировать уникальный перечень всех используемых источников в текстовый файл.

5. ВАЖНЫЕ ЗАМЕЧАНИЯ

5.1. Структура JSON: Приложение строго следует заданной структуре JSON. Редактирование файлов вручную с нарушением структуры может привести к ошибкам.
5.2. Сохранение: Команды меню "Сохранить", "Сохранить как..." и "Отменить изменения" активны только после того, как в редакторе выбран материал или создан новый.
5.3. Копирование (Ctrl + C) и вставка (Ctrl + V) работает ТОЛЬКО на английской раскладке
5.4. Значения необходимо вводить с разделителем ".", при использовании "," значения проигнорируются.
